# BMI Calculator

## Project Overview

This project is a BMI (Body Mass Index) Calculator developed using Python and Tkinter.

The application allows users to enter their weight and height, calculate their BMI, view their BMI category, and receive a health suggestion based on the result.

## Features

* Graphical User Interface (GUI) using Tkinter
* BMI calculation using standard BMI formula
* BMI category classification

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* Health suggestions based on BMI category
* Input validation
* Error handling for invalid inputs
* User-friendly interface

## Technologies Used

* Python
* Tkinter

## BMI Formula

BMI = Weight (kg) / Height² (m²)
