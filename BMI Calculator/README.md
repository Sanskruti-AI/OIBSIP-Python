# BMI Calculator

## Project Overview

This project is an Advanced BMI (Body Mass Index) Calculator developed using Python and Tkinter.

The application allows users to enter their weight and height, calculate their BMI, view their BMI category, receive health suggestions, store BMI records, view history, and analyze BMI trends using graphs.

## Features

* Graphical User Interface (GUI) using Tkinter
* BMI calculation using the standard BMI formula
* BMI category classification

  * Underweight
  * Normal Weight
  * Overweight
  * Obese

* Health suggestions based on BMI category
* Input validation
* Error handling
* BMI history storage using CSV file
* View BMI history
* BMI trend analysis graph
* User-friendly interface

## Technologies Used

* Python
* Tkinter
* CSV
* Matplotlib

## BMI Formula

BMI = Weight (kg) / Height² (m²)

## Advanced Features

### Data Storage

Each BMI calculation is automatically stored in a CSV file for future reference.

### Historical Data Viewing

Users can view previously calculated BMI records through the application.

### Data Visualization

The application generates a BMI trend graph using Matplotlib to help users visualize changes over time.
