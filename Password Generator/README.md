# Password Generator

## Project Overview

This project is an Advanced Password Generator developed using Python and Tkinter.

The application allows users to generate secure passwords based on selected criteria such as letters, numbers, and symbols. It also ensures that the generated password contains at least one selected character type for better security.

## Features

* Graphical User Interface (GUI) using Tkinter
* Password length selection
* Character type customization

  * Letters
  * Numbers
  * Symbols

* Secure password generation with basic rules
* Password strength indicator

  * Weak
  * Medium
  * Strong

* Clipboard copy functionality
* Input validation
* Error handling
* User-friendly interface

## Technologies Used

* Python
* Tkinter
* Random module
* String module

## Security Features

The application improves password security by:

* Ensuring at least one selected character type is included
* Generating random passwords
* Allowing combination of multiple character types
* Encouraging stronger password creation through length and complexity
