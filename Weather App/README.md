# Weather App

## Project Overview

This project is a Weather Application developed using Python, Tkinter, and a Weather API.

The application allows users to enter a city name and view current weather information through a graphical user interface.

## Features

* Graphical User Interface (GUI)
* Weather API integration
* Temperature display
* Feels-like temperature display
* Humidity information
* Weather condition display
* Wind speed information
* Input validation
* Error handling

## Technologies Used

* Python
* Tkinter
* Requests Library
* Weather API

## Learning Outcomes

Through this project, I learned:

* API integration
* JSON data handling
* GUI development using Tkinter
* User input handling
* Error handling
* Working with online data sources
