# OIBSIP Python Internship Projects

This repository contains projects completed during the Oasis Infobyte Python Programming Internship.

## Projects

### 1. BMI Calculator

A GUI-based BMI Calculator developed using Python and Tkinter.

Features:

* BMI Calculation
* BMI Categories
* Health Suggestions
* Input Validation
* Error Handling

### 2. Password Generator

A GUI-based Password Generator developed using Python and Tkinter.

Features:

* Password Length Selection
* Character Type Selection
* Password Strength Indicator
* Clipboard Copy Functionality
* Input Validation

### 3. Weather App

A GUI-based Weather Application developed using Python, Tkinter, and Weather API integration.

Features:

* Current Weather Information
* Temperature
* Feels Like Temperature
* Humidity
* Wind Speed
* Error Handling

## Technologies Used

* Python
* Tkinter
* Requests Library
